| Boundary | Description |

| startup lifecycle -> request handling | If validation is bypassed, invalid config can reach runtime request path. |

| T-1-05 | E | request handlers config access | medium | mitigate | Prohibit per-request settings instantiation; use cached singleton loaded at startup only. |
</threat_model>
