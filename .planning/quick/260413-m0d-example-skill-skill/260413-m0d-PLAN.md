# Quick Task 260413-m0d: Script-backed example skill

## Objective
Add a bundled script to the example skill, expose a safe companion tool that executes that script, and verify the agent can follow skill instructions to invoke it through `/process`.

## Tasks

### Task 1: Add a bundled script-backed skill contract
- Files: `skills/example_skill/SKILL.md`, `skills/example_skill/platform_report.py`
- Action: Expand the skill instructions so they explain when to run the bundled platform report script and what raw output to return. Add a deterministic Python script under the skill directory that prints a unique execution marker plus platform capability lines.
- Verify: `uv run python skills/example_skill/platform_report.py`
- Done: The skill contains both human-readable usage instructions and a deterministic script artifact.

### Task 2: Expose the script through a safe tool and cover it with tests
- Files: `src/tools/examples.py`, `src/tools/__init__.py`, `tests/test_tools.py`
- Action: Add a dedicated tool function that executes only the bundled example skill script via `subprocess.run([...])` without shell interpolation, register it in the shared toolkit, and add regression tests for both tool registration and deterministic output.
- Verify: `uv run pytest tests/test_tools.py -q`
- Done: The toolkit exposes the script execution tool and unit tests prove its contract.

### Task 3: Add an end-to-end chat demo for skill-guided script execution
- Files: `scripts/demos/demo_skill_script.py`
- Action: Create a demo that asks the assistant to use the example skill's bundled script and return the raw output. Assert the response contains the deterministic execution marker and script-produced capability lines so the demo distinguishes real script execution from a generic model paraphrase.
- Verify: `PYTHONPATH="./scripts/demos" uv run python scripts/demos/demo_skill_script.py`
- Done: There is a runnable `/process` demo that verifies the framework can follow skill instructions and execute the bundled script path.
